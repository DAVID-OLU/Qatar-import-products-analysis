# Qatar-import-products-analysis
Qatar Imports Analysis Dashboard: Trade Trends &amp; Product Insights

🎯 Objective
To analyze Qatar’s import data and uncover key insights about top trading partners, product demand, and import trends over time, using interactive dashboards.

## ⚙️ Methodology
1. Data Preparation
  Cleaned dataset and handled missing values
  Standardized country and product names
  Converted date fields into Year & Quarter format

2. Data Modeling
  Structured data for analysis by:
  Country
  Product
  Time (Year/Quarter)

3. Visualization (Power BI)
📌 Page 1: Overview Dashboard
  Top 5 countries by import value
  KPI cards
  Year-over-Year trend by quarter
  Country filter (interactive slicer)

📌 Page 2: Product Analysis
  Top imported products by selected countries
  Import value by product

  📌 Page 3: Trend Analysis
  Value trend of top imported products
  Overall Top 10 imported products - Value vs Quantity


## 📈 Key Insights

-> Top Import Partners
Qatar imports the most from:

China

United States

India

Germany

Italy

-> Import Growth Trend

  Imports show a general upward trend from 2020 to 2024
  Noticeable dip around 2020, likely indicating external disruptions
  Strong recovery and growth post-2021

-> Seasonality (Quarterly Trends)
  Q4 consistently shows higher import values
  Suggests end-of-year demand spikes or strategic stockpiling

-> Top Products
  Certain product categories contribute significantly more to total import value
  Indicates dependency on specific goods

-> High-Growth Products
  Some products show consistent increase in import value over time
  These represent:
  Growing domestic demand
  Potential business/investment opportunities


## 💡 Business Implications
Qatar relies heavily on a few key countries, which may pose supply chain risks
Increasing import trends indicate:
Economic growth
Rising domestic consumption
High-growth products can guide:
Investment decisions
Local production opportunities
Seasonal trends help businesses plan:
Inventory
Logistics
Supply chain operations
